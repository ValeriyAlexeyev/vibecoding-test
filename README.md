# Vibe Coding Academy

Локальное веб-приложение для обучения vibe coding и AI-first разработке: от первых промтов и HTML-страницы до мини-приложения, проверки кода и финального проекта.

## Что внутри

- Главная страница с учебным маршрутом.
- Дашборд ученика с прогрессом.
- Каталог из 10 курсов.
- Уроки с целью, теорией, промтом, кодом, практикой, чек-листом и мини-тестом.
- Практические задания.
- Локальный AI-агент без внешних API: подсказывает следующий урок, помогает с промтами, практикой, ошибками и финальным проектом.
- Библиотека промтов с поиском.
- Чек-листы проверки HTML, CSS, JavaScript и AI-first процесса.
- Мини-тесты с сохранением результатов.
- Раздел "Мои проекты" с заметками.
- Раздел типовых ошибок.
- Финальный проект с критериями готовности.

## Технологии

- React
- Vite
- HTML, CSS, JavaScript
- Локальные JS-данные
- `localStorage` для прогресса, тестов и заметок

Приложение не использует платные API, внешние сервисы и базу данных.

## Запуск одной командой

```bash
pnpm dev
```

После запуска открой адрес из терминала, обычно:

```text
http://127.0.0.1:5173/
```

## Установка зависимостей

Если зависимости еще не установлены:

```bash
pnpm install
```

## Проверка сборки

```bash
pnpm build
```

## Структура проекта

```text
src/
  data/
    course.js        # уроки, промты, чек-листы, задания и проекты
  styles/
    main.css         # светлая адаптивная тема
  main.jsx           # интерфейс приложения и localStorage-логика
index.html
package.json
```

## Как учиться на платформе

1. Открой "Каталог курсов" и посмотри весь маршрут.
2. Перейди в "Уроки", выбери уровень и тему.
3. Прочитай теорию, скопируй пример промта и попробуй практику.
4. Проверь работу по чек-листу.
5. Пройди мини-тест и отметь урок завершенным.
6. Собирай свои идеи в разделе "Мои проекты".
7. В конце выполни финальный проект.

## Данные и приватность

Все учебные данные хранятся в файле `src/data/course.js`. Личный прогресс сохраняется только в браузере пользователя через `localStorage`.
