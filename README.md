# Go Junior Academy

Локальный интерактивный учебник по Go для начинающих: 40 уроков, 320+ примеров кода, 400 тестовых вопросов, 520 практических задач с решениями, 45 мини-проектов, песочница, поиск, прогресс, уровни, достижения, избранное, заметки и чек-лист подготовки к работе.

## Запуск

Откройте `index.html` в браузере. Сервер, база данных, Docker и интернет не нужны.

## Структура

- `index.html` — главная страница.
- `css/style.css` — дизайн, адаптивность, темы и подсветка кода.
- `components/` — navbar, sidebar, progress.
- `js/` — логика приложения, тесты, поиск, LocalStorage и тема.
- `utils/examples.js` — локальные данные для работы без fetch.
- `utils/advanced-content.js` — уровни, достижения, 500+ задач, интервью-разборы, мини-проекты и чек-лист.
- `data/lessons.json`, `data/quiz.json` — JSON-версии данных.
- `lessons/` — 40 страниц уроков.
- `assets/` — локальные SVG-иконки и изображения.

## Как добавлять уроки

Добавьте объект урока в `utils/examples.js`, создайте `lessons/lessonXX.html` по образцу и укажите `data-page="lesson"` плюс `data-lesson-id="XX"`.

## Как добавлять тесты

В массив `questions` добавьте вопрос, 4 варианта, индекс правильного ответа `answer` и объяснение `explanation`.

## Как изменять дизайн

Основные переменные находятся в начале `css/style.css`: фон, панели, текст, линии и синие акценты.

## Хранение данных

Прогресс, баллы, последний урок, избранное, заметки и код песочницы сохраняются в LocalStorage.

## Новые учебные механики

- Уровни: Новичок, Junior, Junior+, Middle.
- Достижения за прохождение ключевых этапов курса.
- График прогресса по всем 40 урокам.
- Избранные уроки с быстрым доступом с главной страницы.
- Личные заметки к каждому уроку.
- Ежедневное случайное задание, выбираемое локально по текущей дате.
- 520 практических задач с подсказками и решениями.
- Разбор типичных технических задач с собеседований.
- 45 реальных мини-проектов: CLI, HTTP, REST, файловые утилиты, JSON, тестирование и конкурентность.
- Чек-лист подготовки к работе Go-разработчиком с сохранением отмеченных пунктов в LocalStorage.
